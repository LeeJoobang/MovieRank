### 작업 개요

- 기간: 2023. 4. 25. - 5. 17.

- 아키텍쳐: MVVM + RxSwift

- 특징

  - TMDB API를 활용하여 인기영화목록을 보여줌.
  - 세부영화 정보 및 필터기능(영화명, 개봉일, 평점) 제공
  - MVVM 패턴을 적용함으로써 수정사항의 영역을 분명하게 함.
  - Pagenation 및 이미지 캐싱을 적용하였음.
  - RxSwift를 사용하여 데이터 비동기처리 적용.
